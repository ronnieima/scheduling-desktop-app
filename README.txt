Title: Appointment Scheduler Desktop App

Purpose: 
The purpose is to create an organized appointment scheduler that pulls data from a database. This program will allow the user to manage all upcoming appointments for customers in different countries and divisions.

Directions for how to run the program:
1. Launch program.
2. Login with user credentials. (Username: admin, Password: admin)
3. You can utilize the add, update, and delete functions for appointments.
4. You can also pull reports or log out.
5. You can also navigate to the customers screen to add, update, and delete.
6. Close the program by clicking the top right X button on the window.

Description of the additional report of your choice you ran in part A3f:
For my additional report, I used the countries and customers tables from the database to get how many customers are in each country. In the table view, the amount of customers are indicated next to each country (US, UK, Canada).
